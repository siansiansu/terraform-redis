# terraform-redis

Managing Redis Enterprise Cloud with Terraform.

## Installation

1. Clone the repo

```bash
git clone git@github.com:siansiansu/terraform-redis.git
```

2. Preview the changes

```bash
terraform plan
```

3. Apply the changes

```bash
terraform apply
```
